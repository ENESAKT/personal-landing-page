# personal-landing-page

My personal landing page — pure HTML & CSS, no frameworks, by design.

**Live:** https://enesakt.github.io/personal-landing-page/

- Semantic HTML5 (`header/nav/main/section/footer`)
- CSS custom properties, Grid project cards, mobile-first
- Dark mode: follows `prefers-color-scheme` and remembers a manual toggle in `localStorage`
- Zero build step — what you read is what ships
